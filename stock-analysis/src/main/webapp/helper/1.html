1​ 概述

1.1​ 简介

移动应用数据统计平台SDK，作为一个应用程序库，随应用程序一起发行，它提供了高效采集分析数据，并在客户端完成初步的数据清洗与封装，传输到server端的数据采用压缩的模式，尽量减少用户的网络数据传输。

目前，SDK提供了三大平台（android、ios、wp）相关的sdk，并采用一致的调用方式，透明的集成到用户的移动应用中。

1、​ 

1.2​ 功能

SDK的实现的主要功能是：

1、​ 采集统计用户使用APP的总体情况（应用的使用次数、频率等）

2、​ 采集统计用户APP使用的页面路径

支持自定义事件，关注事件的发生、结束、及持续时间。

1.3​ 支持平台

目前，SDK支持以下移动应用平台

3​ SDK集成（Android）

3.1​ 导入SDK

1、​ 首先将feinno-sdk-android.1.0.0.jar拷贝到工程lib目录

2、​ 在工程设置中，引用该jar包

3、​ 在需要集成SDK的类中，导入Session会话类

import com.feinno.android.sdk.FeinnoSdkSession;

3.2​ 添加权限

如果需要正确使用SDK，需要在工程文件AndroidManifest.xml中添加一下权限：

<!-- 访问网络状态 -->

<uses-permission android:name="android.permission.ACCESS_NETWORK_STATE"/>

<!-- 访问WIFI状态 -->

<uses-permission android:name="android.permission.ACCESS_WIFI_STATE" />

<!-- 连接网络 -->

<uses-permission android:name="android.permission.INTERNET"/>

<!-- 读取手机状态 -->

<uses-permission android:name="android.permission.READ_PHONE_STATE"/>

3.3​ 初始化SDK

SDK统计用户数据时，需要用户提供注册APP时获取的APPKEY，以及该APP发行时的渠道信息。提供给SDK使用的APPKEY及渠道信息可以通过两种方式设置：

1、​ 在AndroidManifest.xml中设置：

<!-- APPKEY 信息 -->

<meta-data android:name="FEINNO_APP_KEY" android:value="abcd4321"/>

<!-- 渠道信息 -->

<meta-data android:name="FEINNO_APP_CHANNEL" android:value="FEINNO_DP"/>

2、​ 开发者也可以在会话初始化时设置。

FeinnoSdkSession.init(this.getApplicationContext(),"appkey","appChannel");

注意：开发者必须在APP的第一个启动的Activity中的onCreate方法中初始化会话。

//appkey 及 渠道信息在AndroidManifest.xml中配置

FeinnoSdkSession.init(this.getApplicationContext());

//appkey 及 渠道信息在代码中设置 FeinnoSdkSession.init(this.getApplicationContext(),"appkey","appChannel");

3.4​ 会话统计

1、​ 在Activity的OnResume方法中，开始一个会话。

@Override

protected void onResume() {

super.onResume();

FeinnoSdkSession.open();

//其他代码

}

2、​ 在Activity的OnPause方法中，结束一个会话。

@Override

protected void onPause() {

super.onPause();

FeinnoSdkSession.close();

//其他代码

}

注意：在复杂的Activity的继承体系中，会话开始及结束的控制，需要注意顺序。会话的开始一定要在结束之前调用。

3.5​ 页面统计

1、​ 页面开始：

FeinnoSdkSession.pageEnd("pageName");

2、​ 页面结束

FeinnoSdkSession.pageStart("pageName");

注意：页面开始及结束的调用必须在Session的有效期之内才会被记录。即在会话的Open及Close之间调用。

3.6​ 自定义事件统计

1、​ 事件开始：

FeinnoSdkSession.eventEnd("eventName");

2、​ 事件结束

FeinnoSdkSession.eventStart("eventName");

注意：页面开始及结束的调用必须在Session的有效期之内才会被记录。即在会话的Open及Close之间调用。
