Create a scrollbar for AmSerialChart and AmXYChart charts.

Example

var chart = AmCharts.makeChart("chartdiv",{
	...
	"pathToImages": "http://cdn.amcharts.com/lib/3/images/", // required for grips
	"chartScrollbar": {
		"updateOnReleaseOnly": true
	}
});
