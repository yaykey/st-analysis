Extension for AmCharts to export charts to images, check this article to explore more possibilities with that extension.

Example

var chart = AmCharts.makeChart("chartdiv",{
	...
	"amExport": {
		"exportPNG"	: true,
		"imageFileName"	: "amCharts",
		"buttonTitle"	: "Save chart as an image"
	}
});
