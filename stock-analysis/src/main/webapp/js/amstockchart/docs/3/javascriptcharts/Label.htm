Creates a label on the chart which can be placed anywhere, multiple can be assigned.

Example

var chart = AmCharts.makeChart("chartdiv",{
	...
	"allLabels": [
		{
			"text": "Free label",
			"bold": true,
			"x": 20,
			"y": 20
		}
	]
});
