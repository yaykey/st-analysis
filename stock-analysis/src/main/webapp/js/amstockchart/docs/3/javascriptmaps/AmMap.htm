Inheritance: AmMap → AmChart

AmMap is the main class of the AmMap application. Example:

var map = new AmCharts.AmMap();
map.pathToImages = "http://www.ammap.com/lib/images/";
map.dataProvider = {mapVar: AmCharts.maps.worldLow};
map.write("mapdiv");
