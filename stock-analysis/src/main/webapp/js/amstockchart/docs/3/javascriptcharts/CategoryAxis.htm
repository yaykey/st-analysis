Inheritance: CategoryAxis → AxisBase

Extension for AxisBase, gets automatically populated if none has been specified.

Example

var chart = AmCharts.makeChart("chartdiv",{
	...
	"categoryAxis": {
		"gridPosition": "start"
	}
});
